# artmapp

This application uses [Artsy's API](https://developers.artsy.net/) to allow users to rate artworks and artists that are displayed.

## Requirements
* Ruby
* Rails
* Artsy API key

## Installation

* Clone the repo:  
`$ git clone https://github.com/chandracarney/artmapp.git`

* Change into the new directory:  
`$ cd artmapp`

* Bundle install:  
`$ bundle install`

## Running the tests

* Run the tests with rspec:  
`$ rspec`

## Development

* Run the server in your console:  
`$ rails server`

* Visit the app at [http://localhost:3000](http://localhost:3000) to see the splash page.
